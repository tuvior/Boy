# Boy

![Rust](https://img.shields.io/badge/rust-stable-orange)
![Platform](https://img.shields.io/badge/platform-DMG--01-lightgrey)

Work-in-progress **Nintendo Game Boy (DMG-01) emulator in Rust**.

This is a learning-focused project with an emphasis on **CPU correctness** rather than features or performance.

## Implementation

- [x] Cart loading
- [x] Basic MMU
- [x] CPU instructions
- [x] Interrupts
- [x] Timers
- [x] PPU
- - [x] Background
- - [x] Window
- - [ ] Sprites
- [ ] Input
- [ ] Sound
- [ ] Memory banking
